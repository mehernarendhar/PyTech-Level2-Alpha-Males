# Project 1: Student Marks Analyzer (Problem 3)

## Description
This project analyzes student marks using Python.

## Features
- Load dataset using pandas
- Calculate average marks
- Identify highest and lowest scorer
- Generate bar chart and histogram
- No hardcoded data

## Dataset Used
marks.csv  
Source: Kaggle – Students Performance in Exams Dataset

## How to Run
1. Navigate to StudentMarksAnalyzer folder
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python main.py
